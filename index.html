LED Matrix Racing Game
Author:
Andrew Watson (andy@watsons.ch)
Date:
October 2011

Description

This project is a simple game to be played on an 8x8 bi-color LED matrix. It is designed for Robopoly's LED Matrix Module, which was also designed by Andrew Watson. This module features a two-colour LED matrix in the centre of a shield that plugs on top of the PRisme microcontroller platform, surrounded by four tactile switches.

Using the two bottom-most switches, the user moves a "car" (symbolized by a red dot on the bottom line of the matrix) through a scrolling circuit (made of green dots). It is meant primarily as a demo for those who wish to get started using the LED matrix library which ships with this demo.

Controls

Buttons 1 and 4 (bottom left and bottom right) move the car left and right. The functionality of the other buttons has not been decided yet. The options I am considering at the moment are:

  1. Pause button
  2. Increase/decrease game speed
  3. Move car up/down on the screen, thus changing the score and difficulty.

Project location

The code for this demo is currently hosted on github.

The present documentation files are hosted as github pages.

Documentation and design files for the hardware, as well as links to available demos and libraries, can be found at the Robopoly Wiki.
